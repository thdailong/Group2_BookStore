# Group2_BookStore
This is a .NET core MVC web application from Group 2 FPT University

## Run project
In order to run the project, we recommend you install:
- `.NET 5`
- `Mysql`
- `Mysql workbench`

After that, you can run command below on terminal:
`dotnet run`
